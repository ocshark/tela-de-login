🚀 Tecnologias
Este projeto foi desenvolvido utilizando as seguintes tecnologias:

React Native — Framework para desenvolvimento nativo.

Expo — Plataforma para facilitar o desenvolvimento e build.

React Navigation — Navegação entre telas.

Lucide React Native — Biblioteca de ícones (opcional, substitua se usar outra).

Styled Components / Tailwind (NativeWind) — Estilização (especifique a sua).

🛠️ Instalação e Execução
Para rodar este projeto localmente, você precisará do Node.js e do Git instalados.

Clone o repositório:

Bash
git clone https://github.com/seu-usuario/dock.git
cd dock
Instale as dependências:

Bash
npm install
# ou
yarn install
Inicie o servidor de desenvolvimento:

Bash
npx expo start
Abra o app no seu dispositivo:

Use o Expo Go (Android/iOS) para escanear o QR Code no terminal.

Pressione a para abrir no emulador Android.

Pressione i para abrir no simulador iOS.
